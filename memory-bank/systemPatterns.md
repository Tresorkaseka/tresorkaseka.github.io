# System Patterns: Tresor Kaseka Portfolio

**Architecture**: Single-file HTML design (`index.html`) with inlined assets for simplicity and portability.

## Interaction Patterns
- **Text Scramble**: Interactive scrambling effect on hover for the name and potentially other elements.
- **i18n**: Dictionary-based translation system using `data-i18n` attributes and a client-side `setLang` function.
- **Responsive Grid**: Fluid layout system using CSS variables and media queries.

## Visual Design
- **Industrial Hacker Theme**: Uses a grid background, monospace fonts (JetBrains Mono), and "Clash Display" for headings.
- **Color Palette**: Dark base (`#050505`) with amber/yellow accents (`#eab308`).
- **Module System**: Logical separation into modules (Module 01, 02, 03).
