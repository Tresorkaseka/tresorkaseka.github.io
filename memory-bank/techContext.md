# Tech Context: Tresor Kaseka Portfolio

**Core Frontend Environment**: Non-framework HTML/JS/CSS.

## Technologies Used
- **CSS**: Vanilla CSS with custom properties (CSS variables).
- **JavaScript**: Vanilla ES6+ JS for interactions and i18n logic.
- **Fonts**: Google Fonts (JetBrains Mono) and Fontshare (Clash Display).
- **Icons**: SVG-based (inlined or simple symbols).

## Planned/New Stack Inclusions
- **Database**: PostgreSQL.
- **Backend/BAAS**: Supabase.
